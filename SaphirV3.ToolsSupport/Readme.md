# Présentation du projet

Le contexte de ce projet repose sur l'importance croissante de la gestion des environnements Microsoft Dynamics CRM dans les entreprises. Ces systèmes jouent un rôle clé dans la gestion des relations clients, des ventes, et d’autres processus métiers essentiels. Il est donc crucial de s'assurer que ces environnements sont correctement configurés et sécurisés.
L’objectif principal du projet est de développer une application permettant de vérifier l’état et la configuration des environnements Dynamics CRM en se basant sur une checklist détaillée, permettant ainsi de garantir leur conformité, sécurité et performance.


# Structure du projet

Le projet est structurée en deux grandes parties:
- **Frontend** : qui abrite tout les visuels. Il est fait en **VueJs**
- **Backend** : qui abrite tout la logique (API); il est fait avec **flask** (python)
